## a script to automatically search and download paper

This script will search for:
* Google Scholar
* Google

and support downloading pdf from:

* direct pdf link
* dl.acm.org
* ieeexplore.ieee.org
* arxiv.org

using:

* requests
* wget

## Dependencies
* requests
* BeautifulSoup4


## Usage
```bash
./paper-downloadr.py -t "Distinctive image features from scale-invariant keypoints" -d /tmp
```
